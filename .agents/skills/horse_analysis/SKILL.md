# 競馬分析スキル: 実績ベースのスコアリング

このスキルは、オッズや人気に依存せず、馬の過去の競争成績から純粋な身体能力を数値化し、上位3頭を予測するための設計指針です。

## スコアリングの5項目
1.  **スピード**: 過去の走破タイムを距離で正規化した数値
2.  **スタミナ**: 掲示板（5着以内）に入った際の最長距離実績
3.  **瞬発力**: 上がり3ハロン（直線の末脚）の平均タイム
4.  **安定性**: 平均着順のバラツキを抑えた信頼度
5.  **実績**: 勝利数および重賞レースへの出走・入着実績

## 設計原則
- **オッズの排除**: 期待値計算ではなく、能力値の合計（Total Score）で1着(◎)、2着(○)、3着(▲)を決定すること。
- **継続的な更新**: 新しいレース結果（Result）が取得されるたびに、該当馬の `HorseAbility` テーブルを自動更新すること。
- **データの永続化**: スコアリング結果は `keiba.db` に保存し、次回の予想時に再利用・蓄積すること。

## 使用ツール
- `src/analysis/horse_scoring.py`: 能力算出エンジン
- `src/database.py`: `HorseAbility` テーブル定義
